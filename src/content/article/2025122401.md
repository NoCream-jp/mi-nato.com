## はじめに
情報提供をしてくれた友人達への感謝を、まずここへ記しておきたいと思います。

この記事の編集はwebサイトのデザイン、実装と並行しています。適宜修正や大幅な内容の変更が起こる可能性があります。編集した日付はすべての記事で、タイトル部分に記載してあります。

## 個人サイト作成の目的
就職活動に加え、学外の方々との交流を見据えて個人サイトを作成しようと思い立ちました。実績を掲示する場、制作物置き場、近況報告の手段、そしてコミュニケーションの手段として活用していくつもりです。また、初学者の方がwebサイトのデプロイ方法を求めてここを訪れた際に、私の記事がその一助になればと考えています。

## コンセプト
丸みを帯びたシンプルでかわいらしいデザインと見やすさを両立したいというコンセプトの中で試行錯誤した結果、現在の形に落ち着きました。デザインに関しては今後もよりよいかたちを目指して修正を重ねていく予定です。

## 実現方法
本サイトはJavaScriptのフレームワークであるAstroを利用して作成しています。Astroはモダンな静的サイト生成フレームワークで、md(マークダウン)ファイルからページを生成してくれるという強力な特徴を持っています(もちろんほかのSSGフレームワークでも可能なことです)。

環境を簡単に記しておきます。
- **Framework:** Astro (Static Site Generator)
- **Languages:** HTML5, CSS3, JavaScript (ES6+)
- **Hosting:** GitHub Pages
- **CDN / DNS:** Cloudflare
- **Editor:** Cursor

HTMLとCSSを使ってwebサイトを開発した経験のある方は、同じ要領で.astroファイルを編集することでサイトのデザインが可能です。

なおデプロイ方法など詳細は、別の記事にて細かく解説する予定です。
